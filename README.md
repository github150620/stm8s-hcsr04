# stm8s-hcsr04
Use STM8S to control HC-SR04.

## HC-SR04
* An ultrasonic distance meter

## Wires
```
STM8S003F3P3     HC-SR04
+----------+     +-----+
|          |     |     |
|        C4|<--->|Trig |
|          |     |     |
|        D4|<--->|Echo |
|          |     |     |
|       GND|<--->|GND  |
|          |     |     |
+----------+     +-----+
```
